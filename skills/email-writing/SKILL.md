# 비즈니스 이메일 작성

## Overview

한국 비즈니스 맥락에 맞는 이메일을 작성합니다.

## When to Use

- "이메일 써줘", "메일 초안 작성"
- 거래처, 상사, 동료에게 보낼 메일
- 요청, 회신, 안내, 사과 메일

## 기본 구조

```
제목: [간결하고 명확한 제목]

[수신자] 님께,

안녕하세요, [소속] [이름]입니다.

[인사말/본론 도입]

[본문 내용]

[마무리 인사]

감사합니다.

[소속]
[이름] 드림
연락처: [전화/이메일]
```

## 상황별 템플릿

### 요청 메일

```
제목: [업무명] 관련 요청드립니다

OOO 님께,

안녕하세요, [소속] [이름]입니다.

다름이 아니라, [업무명] 관련하여 요청드릴 사항이 있어 연락드립니다.

[요청 내용 상세]

바쁘시겠지만, [기한]까지 회신 주시면 감사하겠습니다.
문의사항이 있으시면 언제든 연락 부탁드립니다.

감사합니다.
```

### 회신 메일

```
제목: RE: [원본 제목]

OOO 님께,

안녕하세요, [소속] [이름]입니다.
문의 주신 내용에 대해 회신드립니다.

[답변 내용]

추가 문의사항이 있으시면 말씀해 주세요.

감사합니다.
```

### 사과 메일

```
제목: [건명] 관련 사과 말씀드립니다

OOO 님께,

안녕하세요, [소속] [이름]입니다.

먼저, [문제 상황]으로 불편을 드린 점 진심으로 사과드립니다.

[원인 설명]

[해결 방안/재발 방지책]

앞으로 이러한 일이 없도록 더욱 주의하겠습니다.
다시 한번 사과드립니다.

감사합니다.
```

## 호칭 가이드

| 관계 | 호칭 | 끝맺음 |
|------|------|--------|
| 상급자 | OOO 부장님/과장님께 | 드림 |
| 동급자 | OOO 님께 | 드림 |
| 하급자 | OOO 님께 | 드림 |
| 외부 | OOO 담당자님께 | 드림/올림 |
| 고객 | 고객님께 | 올림 |

## 톤 조절

### 격식체 (상급자, 외부)
- ~하옵니다, ~주시면 감사하겠습니다
- "검토 부탁드립니다", "회신 부탁드립니다"

### 일반 경어 (동료)
- ~합니다, ~주세요
- "확인해 주세요", "공유드립니다"

## Common Mistakes

| 실수 | 수정 |
|------|------|
| 제목 없음/모호함 | 구체적 제목 필수 |
| 너무 긴 본문 | 핵심만 간결하게 |
| 첨부 누락 언급 | "붙임 파일 확인 부탁드립니다" |
| 연락처 미기재 | 서명에 연락처 포함 |
