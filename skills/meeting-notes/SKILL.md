# 회의록 작성

## Overview

회의 내용을 체계적으로 정리하고 액션 아이템을 도출합니다.

## When to Use

- "회의록 작성해줘", "미팅 노트 정리"
- 회의 내용 텍스트/음성 제공 시
- "어제 회의 내용 요약해줘"

## 회의록 표준 양식

```
# 회의록

## 회의 정보
- **회의명**: [회의 제목]
- **일시**: 2026년 2월 8일 (토) 14:00 ~ 15:30
- **장소**: [회의실/온라인]
- **참석자**: 홍길동 과장, 김철수 대리, 이영희 사원
- **작성자**: [작성자]

---

## 안건

### 1. [안건 1]
**논의 내용**
- [논의 사항 1]
- [논의 사항 2]

**결정 사항**
- [결정 1]
- [결정 2]

### 2. [안건 2]
**논의 내용**
- [내용]

**결정 사항**
- [결정]

---

## 액션 아이템

| No | 항목 | 담당자 | 기한 |
|----|------|--------|------|
| 1 | [할 일 1] | 홍길동 | 02/10 |
| 2 | [할 일 2] | 김철수 | 02/12 |
| 3 | [할 일 3] | 이영희 | 02/15 |

---

## 다음 회의
- **일시**: 2026년 2월 15일 (토) 14:00
- **안건**: [예정 안건]

---

## 비고
- [추가 메모]
```

## 작성 워크플로우

```
회의 내용 수신
    ↓
1. 기본 정보 정리 (일시, 참석자)
    ↓
2. 안건별 분류
    ↓
3. 논의/결정 사항 구분
    ↓
4. 액션 아이템 추출 (담당자, 기한)
    ↓
5. 다음 회의 일정 확인
```

## 핵심 포인트

### 액션 아이템 추출 기준
- **WHO**: 누가 할 것인가
- **WHAT**: 무엇을 할 것인가
- **WHEN**: 언제까지 할 것인가

### 좋은 액션 아이템 예시
```
❌ 나쁜 예: "마케팅 검토"
✅ 좋은 예: "김과장, 2월 마케팅 예산안 검토 후 3/1까지 피드백"
```

## 요약 버전 (간략형)

```
## 회의 요약 (2026.02.08)

**참석**: 홍길동, 김철수, 이영희

### 결정 사항
1. [결정 1]
2. [결정 2]

### TODO
- [ ] 홍길동: [할 일] (02/10)
- [ ] 김철수: [할 일] (02/12)

**다음 회의**: 02/15 14:00
```
