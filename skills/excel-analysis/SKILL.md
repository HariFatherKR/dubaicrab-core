# 엑셀 데이터 분석

## Overview

엑셀/CSV 파일에서 데이터를 분석하고 인사이트를 도출합니다.

## When to Use

- 사용자가 xlsx, xls, csv 파일 업로드
- "데이터 분석해줘", "엑셀 정리해줘"
- 매출, 실적, 통계 데이터 분석 요청

## 분석 워크플로우

```
데이터 수신
    ↓
1단계: 데이터 구조 파악
    ↓
2단계: 기초 통계 (합계, 평균, 최대/최소)
    ↓
3단계: 트렌드/패턴 분석
    ↓
4단계: 인사이트 도출
    ↓
결과 보고
```

## 분석 체크리스트

### 1단계: 구조 파악
- [ ] 전체 행/열 수
- [ ] 컬럼(헤더) 확인
- [ ] 데이터 타입 (숫자, 텍스트, 날짜)
- [ ] 결측치 확인

### 2단계: 기초 통계
- [ ] 합계 (SUM)
- [ ] 평균 (AVERAGE)
- [ ] 최대값/최소값 (MAX/MIN)
- [ ] 개수 (COUNT)

### 3단계: 심화 분석
- [ ] 기간별 트렌드
- [ ] 카테고리별 비교
- [ ] 상위/하위 N개
- [ ] 성장률 계산

## 한국 비즈니스 맥락

### 금액 표기
- 천 단위 콤마: 1,000,000원
- 큰 금액: 10억 원, 1조 원

### 증감률 표현
- 전월 대비 +15.2% 증가

### 기간 표현
- 1Q, 2Q, 3Q, 4Q (분기)
- 1H, 2H (반기)
- FY2026 (회계연도)
