## 1. 로드맵 개요

### 1.1 비전 타임라인

```
2026 Q1          2026 Q2          2026 Q3          2026 Q4
   │                │                │                │
   ├──── MVP ────┤├─ Growth ──┤├── Scale ───┤├─ Enterprise ─┤
   │                │                │                │
   │ 설치 간소화    │ KakaoTalk      │ 기업 관리      │ SLA/지원
   │ HWP 파싱      │ 문서 생성      │ 팀 기능        │ 파트너십
   │ Tauri UI     │ 5개 PoC       │ 20개 계약      │ 해외 진출
```

### 1.2 전략적 우선순위

1. **Survival**: 핵심 기능 완성 (설치, HWP, UI)
2. **Adoption**: 사용자 확보 (KakaoTalk, 온보딩)
3. **Growth**: 기업 시장 진입 (관리 기능, PoC)
4. **Scale**: 확장 (팀 기능, 모바일, 해외)

**RICE 공식**: (Reach × Impact × Confidence) / Effort

### 5.2 우선순위 매트릭스

```
        높은 영향
            │
     P0 ────┼──── P1
            │
─────높은 노력───┼───낮은 노력─────
            │
     P2 ────┼──── P3
            │
        낮은 영향

P0: 인스톨러, HWP 파싱, UI 기본
P1: KakaoTalk, 문서 생성, 관리 기능
P2: 팀 기능, 음성, 워크플로우
P3: 모바일, 플러그인 마켓
```
