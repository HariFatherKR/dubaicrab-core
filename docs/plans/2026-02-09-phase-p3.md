## 🎯 Sprint 3.1: 한글 OCR (RICE 110)

### 목표

- PaddleOCR 통합으로 스캔 문서/이미지에서 한글 텍스트 추출
- 이미지 드래그앤드롭 → OCR → AI 분석 파이프라인 구현

### 기술 선택

| 옵션          | 장점                       | 단점                  | 선택 |
| ------------- | -------------------------- | --------------------- | ---- |
| PaddleOCR     | 한글 정확도 높음, 오프라인 | Python 의존성         | ✅   |
| Tesseract     | 가벼움, 널리 사용          | 한글 정확도 낮음      | ❌   |
| Google Vision | 정확도 최고                | 클라우드 의존, 비용   | ❌   |
| Naver Clova   | 한글 특화                  | API 비용, 온라인 전용 | ❌   |

### 아키텍처

```
┌─────────────────────────────────────────────────────────┐
│                   Image Input                           │
│  (Drag & Drop / Clipboard / File Dialog)                │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│               OCR Processing (Python)                   │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐  │
│  │ Preprocess  │ -> │ PaddleOCR   │ -> │ Post-proc   │  │
│  │ (Resize,    │    │ (Detection  │    │ (Clean,     │  │
│  │  Enhance)   │    │  Recognition)│    │  Format)    │  │
│  └─────────────┘    └─────────────┘    └─────────────┘  │
└─────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────┐
│                 Result Display                          │
│  - OCR 텍스트 표시                                      │
│  - 신뢰도 표시                                          │
│  - AI 분석 연동                                         │
└─────────────────────────────────────────────────────────┘
```
