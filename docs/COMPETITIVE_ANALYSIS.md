> 작성일: 2026-02-08

---

## 1. 시장 개요

### 1.1 타겟 시장

- **ChatGPT 차단 기업**: 대기업, 공공기관, 금융권, 법무법인, 의료기관
- **시장 규모 추정**: 한국 100대 기업 중 약 60% 이상이 외부 AI 서비스 차단
- **Pain Point**: AI 생산성 도구 사용 불가로 인한 업무 효율 저하

### 1.2 시장 트렌드 (2024-2026)

| 트렌드             | 설명                                          |
| ------------------ | --------------------------------------------- |
| 온프레미스 AI      | 데이터 주권 강화로 로컬 LLM 수요 급증         |
| 로컬 LLM 성능 향상 | Llama 3.2, Qwen2.5 등 오픈소스 모델 성능 급등 |
| 하이브리드 배포    | 민감 데이터는 로컬, 일반 쿼리는 클라우드      |
| 한국어 최적화      | 한국어 특화 모델 등장 (Solar, KoAlpaca)       |

## 6. SWOT 분석

### Strengths (강점)

- ✅ 100% 로컬 → 보안 규제 충족
- ✅ HWP 지원 → 경쟁사 대비 유일
- ✅ OpenClaw 기반 → 검증된 아키텍처
- ✅ 멀티 채널 → 유연한 접근성

### Weaknesses (약점)

- ⚠️ 로컬 LLM 성능 한계 (vs 클라우드 GPT-4)
- ⚠️ 초기 설정 복잡도
- ⚠️ 브랜드 인지도 부족
- ⚠️ 기업 지원/SLA 부재

### Opportunities (기회)

- 🔥 ChatGPT 차단 기업 수요 급증
- 🔥 로컬 LLM 성능 급속 발전
- 🔥 데이터 주권 규제 강화
- 🔥 한국어 오픈소스 모델 발전

### Threats (위협)

- ⚡ MS/Google 온프레미스 버전 출시
- ⚡ 네이버/카카오 기업용 확장
- ⚡ 규제 변화 (AI 규제법)
- ⚡ 오픈소스 경쟁자 등장

## 7. 전략적 시사점

### 7.1 단기 전략 (0-6개월)

1. **HWP 지원 강화**: 경쟁 불가능한 영역 확대
2. **설치 간소화**: 원클릭 설치 → 진입 장벽 낮춤
3. **파일럿 고객 확보**: 중견기업 3-5곳 PoC

### 7.2 중기 전략 (6-12개월)

1. **KakaoTalk 연동**: 한국 시장 필수 채널
2. **기업 관리 기능**: 중앙 관리, 사용량 모니터링
3. **커뮤니티 빌딩**: 사용자 피드백 루프

### 7.3 장기 전략 (12개월+)

1. **프리미엄 모델**: Enterprise 지원 패키지
2. **파트너십**: SI/컨설팅사 협력
3. **버티컬 확장**: 법무/의료/금융 특화

## 8. 결론

Dubai Crab은 **"로컬 + HWP + 한국 최적화"** 조합으로 차별화된 포지션을 가진다.

**핵심 경쟁 우위**:

1. 한국 기업 규제 환경 완벽 대응 (로컬 + 오프라인)
2. HWP 지원으로 경쟁 불가능한 니치 점유
3. 멀티 채널로 유연한 업무 통합

**주요 도전**:

1. 로컬 LLM 성능 vs 클라우드 GPT-4 격차 극복
2. 기업 영업 및 지원 체계 구축
3. 브랜드 인지도 확보
